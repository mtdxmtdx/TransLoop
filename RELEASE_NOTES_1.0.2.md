# TransLoop 1.0.2

## 定位

TransLoop 1.0.2 是面向 Windows 桌面划词翻译与截图 OCR 翻译的修复更新，重点解决截图翻译设置加载失败，并调整本地 Provider 请求频率策略。

## 修复与改进

- 截图窗口和划词悬浮窗可以安全触发旧版凭证迁移；敏感凭证值仍只在 Rust 原生侧处理，API Key 保存、删除和状态操作仍限制在主窗口。
- 普通文本翻译本地最多每 1 秒 1 次，即 60 次/分钟。
- 截图视觉翻译和 OCR 识别本地最多每 2 秒 1 次，即 30 次/分钟。
- Provider 连通性测试本地最多每 5 秒 1 次，即 12 次/分钟。
- 不同 Provider 和操作类型分别计数；全局同时进行的请求仍最多为 2 个。
- 新增频率策略和即时重复请求拦截测试。

## 升级说明

- 可从 `1.0.1` 直接安装升级到 `1.0.2`。
- 设置、Credential Manager 中的 Provider API Key、历史、缓存和快捷键会继续保留在本机。
- 应用内更新不做静默安装；验证成功后才启动 NSIS 安装程序。

## 安装包

- Windows x64 NSIS 安装包：`TransLoop_1.0.2_x64-setup.exe`
- 更新校验资产：`SHA256SUMS`、`SHA256SUMS.sig`
