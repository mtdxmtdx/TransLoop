# TransLoop 0.9.0 RC

## 定位

0.9.0 是正式版候选版本，重点是稳定性回归、错误提示统一、文档和发布验证。不新增大功能，不恢复浏览器插件路线。

## 改进

- 统一版本号到 `0.9.0`。
- 主窗口、划词翻译、截图翻译、Provider 测试、更新下载、设置导入和诊断包导出会显示更可读的错误提示。
- 增强 Provider 错误归类，覆盖超时、网络失败、限流、HTTP 4xx/5xx、缺 API Key、模型不支持图片和空响应。
- 补充 RC 发布检查清单，覆盖构建、安装升级、Provider、OCR、隐私和 GitHub Release 验证。

## 隐私说明

- 诊断日志和诊断包不包含 API Key、原文、译文、截图或缓存译文正文。
- 设置导出不包含 keyring 中的 API Key。
- 用量日志只保存请求元数据、状态、耗时、字符/token/费用估算和错误摘要。

## 已知限制

- 0.9.0 不提供静默自动更新，只支持检查新版本、下载安装包并启动安装程序。
- OCR 模式 C 依赖本机已安装 Tesseract OCR 和对应语言包。
- Provider 成本统计为本地估算值，不作为真实账单依据。

## 安装包

- Windows x64 NSIS 安装包：`TransLoop_0.9.0_x64-setup.exe`
