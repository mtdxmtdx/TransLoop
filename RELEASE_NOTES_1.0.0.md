# TransLoop 1.0.0

## 定位

TransLoop 1.0.0 是正式稳定版，面向 Windows 桌面划词翻译与截图 OCR 翻译场景发布。不新增大功能，重点是把 0.9.0 RC 验证通过的能力整理为正式版。

## 核心能力

- 划词翻译：任意应用选中文字后按快捷键，在光标旁查看译文。
- 截图翻译：框选屏幕区域，使用 OCR A/B/C 识别并翻译。
- 多 Provider：支持 DeepSeek、OpenAI 兼容服务、Qwen、Grok、MiniMax、Claude、Gemini。
- 自动降级：主 Provider 或模型失败时可尝试备用模型 / Provider。
- 历史、缓存、用量：本地保存翻译历史，短期复用重复翻译，并记录不含正文的用量元数据。
- 产品化工具：首次引导、更新检查、诊断包导出、设置导入 / 导出。

## 文档与隐私

- README 已改为正式版用户文档，包含下载、快速开始、OCR 模式选择、隐私、本地数据和 FAQ。
- 新增 `PRIVACY.md`，说明 API Key、历史、缓存、用量、诊断包和 Provider 请求的数据边界。
- 诊断包和设置导出不包含 API Key、原文、译文、截图或缓存译文正文。

## 升级说明

- 可从 `0.9.0 RC` 直接安装升级到 `1.0.0`。
- 设置、keyring API Key、历史、缓存和快捷键会继续保留在本机。
- 如更新下载失败，可从 GitHub Release 页面手动下载安装包。

## 已知限制

- OCR 模式 C 需要本机安装 Tesseract OCR 和对应语言包。
- 截图模式 A 需要支持视觉输入的 Provider / 模型。
- 应用内更新不做静默安装，只会下载安装包并启动安装程序。
- 用量统计是本地估算值，不作为服务商真实账单依据。

## 安装包

- Windows x64 NSIS 安装包：`TransLoop_1.0.0_x64-setup.exe`
